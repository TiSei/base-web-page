Chart-Attributes

tagdescription
bwp-chart-typeselect charttype, in time supported:
  • piechart
  • columnchart
bwp-chart-datajson-coded chart data, that will shown.
The format is defined by the used charttype
bwp-chart-no-textallows to disable chart description except legend
default, if not set: false
bwp-chart-no-hoverallows to disable chart hover effects includes legend
default, if not set: false
bwp-chart-baselineallows to set baseline value in scaled charts
default, if not set: Minimum of Values - 10%
bwp-chart-legendallows to set legend value in charts
default, if not set: not shown
