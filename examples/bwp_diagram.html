Charts - Attributes

tagdescription
bwp-chart-typeselect charttype, in time supported:
  • piechart
  • columnchart
  • linechart
bwp-chart-labels
optional		allows to disable chart description except legend
default, if not set: false
bwp-chart-hover
optional		allows to disable chart hover effects includes legend
default, if not set: false
bwp-chart-legend
optional		allows to show legend
default, if not set: not shown
bwp-chart-legend_width
optional		force space of legend
default, if not set: 200

Data - Attributes

tagdescription
bwp-chart-datajson-coded values to display
automatic set by API-Requests
bwp-chart-data-source
optional		url to call API for dynamic values
default, called once by loading script
bwp-chart-recall
optional		seconds until repeats call API

Scaled Charts - Attributes (Columncharts, Linecharts)

tagdescription
bwp-scaled-chart-baseline
optional		allows to set baseline value in scaled charts
default, if not set: Minimum of Values - 10%
bwp-scaled-chart-x_width
optional		allows to set width of chart (x-axis) in scaled charts
default, if not set: 150 (square representation)
bwp-scaled-chart-y_width
optional		allows to set width of chart (y-axis) in scaled charts
default, if not set: 150 (square representation)
