Example - SlideShow

Attributes

tagdescription
data-bwp-autoplay

optional

int value represents the seconds until the next slide is automatically displayed.
data-bwp-source

optional

API URL to request dynamic sideshow data
If not set, the slides must be defined manually
data-bwp-slide-type

optional

defines the HTML tags on which the crawled slides are generated
supported:
  • iframe (default)
  • img
data-bwp-recall

optional

int value represents the minutes until the data is updated by calling the API again
only usable by data-bwp-source

Example:
<div class="bwp-slideshow" data-bwp-autoplay="10" data-bwp-source="http://127.0.0.1:8000/examples/sideshow_data.json.example" data-bwp-recall="1" data-bwp-slide-type="img">
Creates a slideshow that retrieves a JSON-encoded list of image URLs from the API once per minute and displays them as images, alternating every 10 seconds.
